# Проект: Место

## Описание
Интерактивная страницей, куда можно добавлять фотографии, удалять их и ставить лайки.

## Стек:
* HTML
* CSS
* JavaScript
* Webpack

## Технологии
* Flexbox 
* Grid
* Адаптивность интерфейса для устройств с различной шириной экрана
* БЭМ-разметка
* Структура файлов Nested
* Реализация модальных окон
